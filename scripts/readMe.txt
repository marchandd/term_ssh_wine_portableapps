Be sure to be through GUI SSL with Root user to apply this install.

First step, apply script to set for having exploit Wine launcher:
/usr/local/sbin/_installFirst_winetricksOptions.sh

All software could be installed separatly. Alias is the best way to launch software after it install was made.

Installing Todolist software with his alias, running:
/usr/local/sbin/install_TodolistPortable.sh

Installing Freecommander software with alias, running:
/usr/local/sbin/install_FreecommanderPortable.sh
Installing Freecommander alias, running:
/usr/local/sbin/postInstall_AliasForFreecommanderPortable.sh

Installing Notepad++ software with alias, running:
/usr/local/sbin/install_NotepadPortable.sh
Installing Notepad++ alias, running:
/usr/local/sbin/postInstall_AliasForNotepadPortable.sh

Installing Winmerge software with alias, running:
/usr/local/sbin/install_WinmergePortable.sh
Installing Winmerge alias, running:
/usr/local/sbin/postInstall_AliasForWinmergePortable.sh

Installing AntRenamer software with alias, running:
/usr/local/sbin/install_AntrenamerPortable.sh
Installing AntRenamer alias, running:
/usr/local/sbin/postInstall_AliasForAntrenamerPortable.sh
